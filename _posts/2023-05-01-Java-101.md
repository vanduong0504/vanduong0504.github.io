## Primitive Types
 - **Number**: byte(8) > short(16) > int(32) > long(64)
 - **Floating Point Number**: float(32) > double(64)
 - **Char**: 16 bits, one character, ''
 - **Boolean**: should start varibale with prefix 'is', 'has' (isMarried, hasChildren)
